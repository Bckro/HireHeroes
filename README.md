# HireHeroes

HireHeroes is a console application developed as part of an object-oriented programming exercise in C# using Visual Studio. The application allows you to manage a guild of heroes and assistants for hire.

## Functionalities

- Guild Management: Add, remove, and display heroes and assistants in the guild.
- Hiring Heroes: Hire available heroes from the guild.
- Releasing Heroes: Release hired heroes back to the guild.

## Getting Started

To run the HireHeroes application, follow these steps:

1. Clone the repository or download the source code files.
2. Open the project in Visual Studio.
3. Build the project to compile the application.
4. Run the application from the console.
5. Follow the on-screen instructions to navigate through the application and use its functionalities.
